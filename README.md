Minimal starter project to implement Strava login (OAuth).

## Quick start

1. Copy `.env.example` -> `.env` and fill in `STRAVA_CLIENT_ID`, `STRAVA_CLIENT_SECRET`, and `SESSION_SECRET`.
2. Install dependencies:

   npm install

3. Run in development:

   npm run dev

4. Open `http://localhost:3000` and click **Log in with Strava**.

Notes:
- This is intentionally minimal: only authentication and a small profile display are implemented.
- Next steps: persist sessions in DB, fetch activities, build review UI.
